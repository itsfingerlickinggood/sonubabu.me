Philosophy: No Frameworks, No Bloat

This website is a deliberate rejection of the modern frontend stack. No React, no Next.js, no Webpack, no npm—just plain HTML, CSS, and vanilla JavaScript served as static files. The decision wasn’t born from ignorance of those tools (several projects in this portfolio use them extensively) but from a conviction that a personal website should embody the simplicity it advocates. Every page loads in under a second on a 3G connection. There are zero runtime dependencies. The entire site fits in a few hundred kilobytes, and every byte earns its place.

Canvas Animations

Each page features a unique 2D canvas animation rendered in the hero section, driven by a lightweight visualization engine in viz.js. The animations are generative—particle systems, flow fields, geometric compositions, emergent patterns—each tied to a data-viz attribute on the hero element. They respond to the active color theme, pulling palette values from CSS custom properties so dark mode and light mode each produce a distinct visual character. The renderer uses requestAnimationFrame with automatic throttling to cap at 30 fps on low-power devices, and pauses entirely when the canvas scrolls out of the viewport via an IntersectionObserver. The goal was to give each page a living, breathing header without the weight of WebGL or a canvas library.

Design and Typography

The typographic system pairs DM Sans for interface elements and navigation, Newsreader for long-form body text, and JetBrains Mono for code snippets. This combination balances geometric clarity with the warmth of a serif reading face—a nod to the academic style of publications like distill.pub, where information density coexists with visual calm. The color system is built entirely on CSS custom properties scoped under [data-theme] selectors, making theme switching instantaneous with zero flash of unstyled content (a FOUC prevention script in the <head> sets the theme attribute before the first paint). Spacing follows a consistent scale derived from a base unit, and the layout uses CSS Grid for page structure with no media-query breakpoints below tablet width—the fluid type and spacing scale handle most responsive needs intrinsically.

Features and Craft Details

Dark and light modes persist across sessions via localStorage, with the initial theme respecting the user’s OS preference through prefers-color-scheme. The research pages use interactive data visualizations rendered inline alongside academic-style prose. The 404 page features a retro airport flip-clock animation that displays the error code with mechanical character-splitting transitions. Navigation is minimal—five links and a theme toggle—because the site’s architecture is flat enough that deep menus would add complexity without aiding discovery. The entire project is a reminder that the web’s core technologies are remarkably capable on their own, and that sometimes the most impressive engineering is knowing what not to include. Inspired by the restraint of ibelick.com and the clarity of distill.pub, this site aims to be fast, readable, and quietly distinctive.
